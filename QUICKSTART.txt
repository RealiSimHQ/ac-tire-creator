AC TIRE CREATOR - QUICK START
==================================================

LOCATION:
/home/paddy-bot/.openclaw/workspace/tyre-creator/app/

HOW TO USE:
1. Open index.html in any web browser (Chrome, Firefox, Edge, Safari)
2. Drag & drop your car's car.ini file (or ZIP containing it)
3. Select tire widths (front and rear)
4. Choose tire pack (Street, Competition, or both)
5. Click "Generate Tire Pack"
6. Extract the downloaded ZIP to your AC car's data folder

FILES INCLUDED:
- index.html         → Main application
- style.css          → Dark theme styling  
- tire-data.js       → 17 tire compound definitions
- physics-generator.js → Physics calculation engine
- app.js             → UI logic and ZIP generation
- README.md          → Full documentation
